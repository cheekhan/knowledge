# 欢迎使用 KnowledgeBase 🎉

这是一款**本地优先**的综合桌面知识库软件。

## 快速上手

1. **写笔记** — 在文件树中点击笔记即可编辑，支持 Markdown 语法
2. **读 PDF** — 打开 PDF 后可缩放、翻页、添加书签和标注
3. **边读边记** — 阅读 PDF 时点击工具栏"**+ 笔记**"创建绑定笔记，左侧阅读右侧编辑
